# CIS Website - New Look

This workspace provides a new look for the CIS website (reference: `instrumentation-solutions.com`, live: https://rajan.instrumentation-solutions.com/).

## Included files

- `index.html` — homepage with hero, products, services, brochures, newsletters, events, videos, clients, and contact sections.
- `about.html` — company overview and history.
- `product.html` — product category page with brochure resources.
- `service.html` — service offerings and technical document links.
- `brochures.html` — downloadable brochure listings.
- `newsletters.html` — newsletter publications.
- `attended.html` — attended events and conference summaries.
- `upcoming.html` — upcoming industry participation.
- `video.html` — technical video overview page.
- `clients.html` — client list and logos.
- `contact.html` — contact details and an inquiry form placeholder.
- `styles.css` — shared responsive styling.
- `script.js` — mobile menu toggle and contact form placeholder behavior.
- `assets/` — local image assets used by the clone.

## How to use

1. Open `index.html` in a browser.
2. Review the pages and update text, images, or links as needed.
3. Add backend form handling later in `contact.html` when ready.

## Notes

This clone includes static content and assets that mirror the original CIS site. The contact form is a frontend placeholder and requires backend integration to submit data.
